记录每次请求的日志，使用官方开发的第三方库 morgan



使用中间件进行文件上传。

使用官方提供的第三方中间件 multer

```js
const upload = multer({
  dest: './uploads' // 保存文件的路径
})

upload.single('[filed-name]') // 返回一个读取单个文件的中间件。

// 在下一个中间件中，使用 res.file 查看文件的信息
res.file 
```

给上传的文件，自定义名称。

```js
const upload = multer({
  destination(res, file, callback) {
    //...
  },
  filename(res, file, callback) {
    //...
  }
})
```

上传多文件。

```js
upload.array()
```
